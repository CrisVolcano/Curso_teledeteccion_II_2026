# GF0662 - Teledetección

**Universidad de Costa Rica**  
**Ciclo lectivo:** 2026  
**Docente:** Cristian Aguilar-Barboza

## Objetivo

Este repositorio centraliza los materiales del curso **GF0662 - Teledetección** para facilitar el acceso del estudiantado al programa, lecturas, recursos de apoyo, laboratorios, scripts y datos de práctica.

El curso introduce fundamentos, herramientas y aplicaciones de la teledetección para el análisis del territorio mediante sensores remotos, imágenes satelitales, plataformas en la nube y métodos de procesamiento digital de imágenes.

## Estructura del repositorio

- `docs/`: programa del curso, documentos académicos e instrucciones generales.
- `extras/`: guías de preparación para plataformas y cuentas necesarias, como Google Earth Engine, NASA Earthdata y Copernicus Data Space Ecosystem.
- `resources/`: bibliografía obligatoria, bibliografía complementaria y recursos externos.
- `labs/`: prácticas, ejercicios guiados y materiales de laboratorio.
- `scripts/`: código de apoyo para procesamiento, análisis y automatización.
- `data/`: datos de ejemplo o insumos pequeños para prácticas.
- `assets/`: recursos visuales utilizados por la página del curso.
