# GF0662 - Teledeteccion

**Universidad de Costa Rica**  
**Ciclo lectivo:** 2026  
**Docente:** Cristian Aguilar-Barboza

## Objetivo

Este repositorio centraliza los materiales del curso **GF0662 - Teledeteccion** para facilitar el acceso del estudiantado al programa, lecturas, recursos de apoyo, laboratorios, scripts y datos de practica.

El curso introduce fundamentos, herramientas y aplicaciones de la teledeteccion para el analisis del territorio mediante sensores remotos, imagenes satelitales, plataformas en la nube y metodos de procesamiento digital de imagenes.

## Estructura del repositorio

- `docs/`: programa del curso, documentos academicos e instrucciones generales.
- `extras/`: guias de preparacion para plataformas y cuentas necesarias, como Google Earth Engine, NASA Earthdata y Copernicus Data Space Ecosystem.
- `resources/`: bibliografia obligatoria, bibliografia complementaria y recursos externos.
- `labs/`: practicas, ejercicios guiados y materiales de laboratorio.
- `scripts/`: codigo de apoyo para procesamiento, analisis y automatizacion.
- `data/`: datos de ejemplo o insumos pequenos para practicas.
- `assets/`: recursos visuales utilizados por la pagina del curso.

## Pagina del curso

La pagina principal del curso esta disponible en este repositorio mediante GitHub Pages y funciona como punto de entrada para consultar los materiales publicados.
